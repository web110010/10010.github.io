Joys of a Tiling Window Manager

I have been using tiling windows in some form or the other for a good part of 5 years now. Be it via GNOME extensions, Kwin tiling, Pop!OS tiling mode, i3, sway, bspwm, awesome. Finally, I have set my self comfortably on sway. What is a tiling window manager? A tiling window manager in simple terms is a organisation system, which organizes windows in a non overlapping way.
